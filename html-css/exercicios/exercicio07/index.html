Testando formas de Letras

Abaixo será mostrado formas:

Eu vou mostrar uma frase em negrito, usando a tag <b>(sem semantica, apenas na forma em negrito).

E agora vou estar deixando uma frase de destaque em negrito, com a tag <strong>(com semantica, com significado e nao apenas em negrito.).

Itálico / Enfase

E tambem vou deixar uma frase em itálico (sem semantica).

Vou deixar mais uma frase em itálico(semantica), usando a tag <em>.

Texto excluido

Posso deixar um texto excluido, para podermos ler mas nao darmos muita importancia para ele, usando a tag <del>

Texto inserido

Posso tambem deixar algum texto inserido para dar enfase em alguma frase.

Tambem é possivel fazer sublinhado com a tag <U>(sem semantica).

Texto marcado

Posso marcar algum texto quando quiser dar enfase em algum texto, com a tag <Mark>.

Texto pequeno e texto grande

Posso deixar alguma frase grande com a tag <big>.

E tambem posso deixar algum texto pequeno, com a tag <Small>.

Texto sobrescrito

Posso sobrescrever essa frase H20, com a tag <sub>.p>

E tambem sobrescrever essa frase tambem X20+10, com a tag <sup>.
